# 移除"查看原文件"链接设计文档

**日期：** 2026-06-22

**目标：** 移除 website 下所有 md 文件中的"查看原文件"部分及其链接，因为在静态网站中无法直接链接到本地文件系统。

## 背景

问题：website 下的 md 文件中，每个文件都有"查看原文件"部分，包含指向 `../src/...` 的链接。在网页端点击后，提示"未找到"，因为静态网站无法直接链接到本地文件系统。

## 设计方案

### 修改内容

**需要移除的部分：**
- "## 查看原文件" 标题
- 文件路径链接列表（如 `- 源码位置: [...]`）

**保留的部分：**
- "## 运行方式" 部分保持不变

### 示例

**修改前：**
```markdown
## 查看原文件

- 源码位置: [`src/01-setup/01-simple-llm.ts`](../src/01-setup/01-simple-llm.ts)

## 运行方式

```bash
npm run dev src/01-setup/01-simple-llm.ts
```
```

**修改后：**
```markdown
## 运行方式

```bash
npm run dev src/01-setup/01-simple-llm.ts
```
```
